# ArrowRush (Prototype)
Tower Defense Game

# Game design
*Place defensive structures

*Defend an objective

*Waves of enemies

////////////////////////////////////////////////////////////////////////////////////////////////////

Player Experience:

*tactical / strategic

Core Mechanic:

*Using limited resources,strategically place towers to stop
enemies from reaching their goal.

Core game loop:

*Survive against waves of enemies for as long as possible!
